# 🏥 Medical Cost Predictor — полная инструкция для чайников

## ❓ Что это такое?

Это программа, которая предсказывает, сколько денег потратит человек на медицинскую страховку, 
зная его возраст, вес, количество детей и курит ли он.

Программа умеет:
1. **Обучаться** — смотреть на данные и находить закономерности
2. **Предсказывать** — для новых людей говорить цену
3. **Показывать красиво** — через сайт с графиком

---

## 📁 Что за файлы?

| Файл | Что делает | Для кого |
|------|------------|----------|
| `insurance.csv` | Данные о людях (возраст, вес, цена и т.д.) | Компьютер читает их |
| `medical_cost.py` | Обучает модель (один раз) | Ты запускаешь первый |
| `medical_model.pkl` | Готовая модель (создаётся сама) | Компьютер сохраняет |
| `features.pkl` | Список признаков (создаётся сам) | Компьютер сохраняет |
| `predict_medical.py` | Предсказание в консоли | Ты вводишь данные, получаешь цену |
| `app.py` | Веб-сайт | Открываешь в браузере |
| `requirements.txt` | Список библиотек | Чтобы всё установилось |
| `README.md` | Эта инструкция | Ты читаешь её |

---

## 🚀 ПЕРВЫЙ ШАГ — Установка Python

Если Python НЕ установлен:
1. Зайди на сайт: https://www.python.org/downloads/
2. Нажми жёлтую кнопку "Download Python 3.13"
3. Запусти скачанный файл
4. **ОБЯЗАТЕЛЬНО** поставь галочку "Add Python to PATH"
5. Нажимай "Install Now"

Проверь, что Python установился:
```bash
python --version
```
Должно показать: `Python 3.13.7` или что-то подобное.

---

## 🚀 ВТОРОЙ ШАГ — Скачай все файлы

Скачай все файлы проекта в одну папку:
- `insurance.csv`
- `medical_cost.py`
- `predict_medical.py`
- `app.py`
- `requirements.txt`
- `README.md`

Положи всё в папку, например: `C:\Users\Твое_Имя\Desktop\medical_project`

---

## 🚀 ТРЕТИЙ ШАГ — Открой терминал

**Windows:**
1. Зажми `Win + R`
2. Напиши `cmd` и нажми Enter
3. Перейди в папку с проектом:
```bash
cd C:\Users\Твое_Имя\Desktop\medical_project
```

**Mac/Linux:**
1. Открой Terminal
2. Перейди в папку:
```bash
cd /Users/Твое_Имя/Desktop/medical_project

## 🚀 ЧЕТВЁРТЫЙ ШАГ — Создай виртуальное окружение

Это нужно, чтобы не испортить другие программы на компьютере.

```bash
python -m venv venv
```

Ничего страшного не произойдёт, просто появится папка `venv`.

---

## 🚀 ПЯТЫЙ ШАГ — Активируй виртуальное окружение

**Windows (PowerShell):**
```bash
.\venv\Scripts\Activate.ps1
```
Если выдаёт ошибку, напиши:
```bash
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
И повтори команду.

**Windows (CMD):**
```bash
venv\Scripts\activate
```

**Mac/Linux:**
```bash
source venv/bin/activate
```

**Проверка:** В начале строки должно появиться `(venv)`.

## 🚀 ШЕСТОЙ ШАГ — Установи библиотеки

Одной командой установи всё нужное:
```bash

```

Подожди 1-2 минуты, пока всё скачается.

## 🚀 СЕДЬМОЙ ШАГ — Запусти обучение (первый раз)

```bash
python medical_cost.py
```

Что произойдёт:
- Компьютер прочитает данные из `insurance.csv`
- Научится искать закономерности
- Сохранит модель в `medical_model.pkl`
- Сохранит список признаков в `features.pkl`
- Покажет график (закрой его, чтобы продолжить)

**Важно:** Это делается **один раз**. Потом модель уже готова.

---

## 🚀 ВОСЬМОЙ ШАГ — Проверь, что модель сохранилась

В папке должны появиться два новых файла:
- `medical_model.pkl`
- `features.pkl`

Если они есть — всё хорошо.

## 🚀 ДЕВЯТЫЙ ШАГ — Запусти консольное предсказание

```bash

```

Теперь вводи данные:

ИМТ: 28
Детей: 2

```bash
python app.py
```

Должно появиться:
```

```

```

```

Что ты увидишь:
- **Сверху** — статистика (точность модели, количество строк)
- **Форма** — поля для ввода (возраст, ИМТ, дети, курит)
- **Таблица** — первые 10 строк данных

## 🎯 Как пользоваться сайтом?

1. Введи возраст (от 18 до 100)
2. Введи ИМТ (от 10 до 60)
3. Введи количество детей (от 0 до 10)
4. Выбери "Yes" или "No" (курит или нет)
5. Нажми кнопку **Predict**

Внизу появится зелёная карточка с предсказанной ценой.

## ❓ Частые проблемы и как их решить

### Ошибка: `'pip' is not recognized`
**Решение:** Установи Python заново и поставь галочку "Add Python to PATH"

### Ошибка: `No module named 'pandas'`
**Решение:** Забудь установить библиотеки. Напиши `pip install -r requirements.txt`

### Ошибка: `FileNotFoundError: medical_model.pkl`
**Решение:** Сначала запусти `python medical_cost.py`, чтобы создать модель

### Ошибка: `ValueError: 8 columns passed, passed data had 10 columns`
**Решение:** Неправильно ввёл данные. Перезапусти `predict_medical.py` и введи всё заново.

### Сайт не открывается в браузере
**Решение:** Проверь, что в терминале написано `Running on http://127.0.0.1:5000`. 
Если нет — подожди 10 секунд.

### График не показывает
**Решение:** Закрой и снова открой браузер. Может, картинка долго грузится.
